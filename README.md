# :information_source: How To Use

```bash

# Clone this repository

$ git clone https://github.com/luisrodriguesds/blog-js.git

# Go into the repository

$ cd blog

# Install packages

$ npm install

# Configure .env with your database information

$ gedit .env

# Run the migrations

$ adonis migration:run

# Run start and enjoy

$ adonis serve --dev

# Enjoy!

```
